# toolchain-parallax_p8x32a
toolchain-parallax_p8x32a is a CMake toolchain for the Parallax P8X32A Propeller
microcontroller.

## git Hooks
To install this repository's git hooks, run the `install` script which is located in the
`hooks` directory.
```
./hooks/install
```
